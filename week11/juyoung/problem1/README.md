# 문제 개요

문제 : [별자리 만들기](https://www.acmicpc.net/problem/4386)

문제 분류 : 그래프 이론

# 문제 풀이

**크루스칼 알고리즘**을 사용했다.

1. 모든 별 사이의 거리를 구해서 벡터 `lines`에 저장한다.
2. `lines`를 오름차순 정렬한다.
3. `lines`의 원소를 하나씩 꺼내서 MST를 만든다.
   - Union-Find 알고리즘을 사용한다.
   - 두 별의 부모가 같지 않은 경우에만 Union 하고 `answer`에 두 별 사이의 거리를 더한다.
4. `answer`를 출력한다.
