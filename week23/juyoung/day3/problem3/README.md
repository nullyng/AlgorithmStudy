# 문제 개요

문제 : [색종이 만들기](https://www.acmicpc.net/problem/2630)

문제 분류 : 분할정복

# 문제 풀이

전체 종이의 사분면을 재귀적으로 탐색하며 자를 색종이의 개수를 구한다.

(0, 0)부터 `N`만큼 탐색  
→ 자를 색종이가 없다면 (0, 0), (0, `N`/2), (`N`/2, 0), (`N`/2, `N`/2)에서 각각 `N`/2만큼 탐색  
→ 또 없다면 현재 범위에서 사분면으로 나눠서 (`N`/2)/2만큼 탐색  
→ ...
